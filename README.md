# NexaTag AI+ 🔐

Smart RFID-based Access Control System using ESP32 and RC522 RFID module.

---

## 🚀 Features
- RFID authentication
- RGB LED indication
- Buzzer alerts
- ESP32-based system
- PCB-ready architecture
- IoT-ready design

---

## 🛠️ Components Used
- ESP32 DevKit
- RC522 RFID Module
- RGB LED
- Buzzer
- Resistors

---

## 🔌 Circuit Connections

| RC522 | ESP32 |
|---|---|
| SDA | GPIO5 |
| SCK | GPIO18 |
| MOSI | GPIO23 |
| MISO | GPIO19 |
| RST | GPIO27 |
